SOP Generator

Screenshots to documentation, automatically

Watch your Greenshot folder and auto-generate step-by-step IT documentation. Perfect for MSPs building client runbooks and internal KB articles.

View on GitHub Get Started

How It Works

📸

1. Capture

Work through your process, hitting Greenshot hotkeys at each step. Region, window, or full-screen captures all work.

⏱️

2. Pause

Stop for 5 seconds. The watcher detects idle time and prompts you for a documentation title.

📝

3. Generate

Markdown SOP auto-generated with numbered steps, image references, and placeholder captions.

📋

4. Paste

Content auto-copied to clipboard. Paste directly into Halo, ITGlue, or your PSA of choice.

🤖

AI Captions (Planned)

Azure OpenAI Vision integration coming soon. Auto-generate step descriptions from screenshots.

⚡

Lightning Fast

No cloud upload delays. Local processing with Ditto clipboard history for the win.

Installation

Quick Start

# Clone the repository git clone https://github.com/Midtown-Technology-Group/sop-generator.git cd sop-generator # Install Python dependencies pip install requests # Configure Greenshot path in sop_generator.py # Then run: python sop_generator.py --watch

Windows (Easy Mode)

# Double-click to start watching double-click start_watch.bat # Or use PowerShell .\Start-SOPGenerator.ps1 -Watch # First run: edit the CONFIG section # Set your Greenshot folder path

Example Workflow

Generated Markdown
# How to Reset Office 365 Password **Created:** 2025-01-15 14:32 **Screenshots:** 5 --- ## Step 1 **Action:** [TODO: Navigate to Azure AD admin center] ![Step 1](./assets/step_01_2025-01-15_14-30-22.png) --- ## Step 2 **Action:** [TODO: Click 'Users' in left navigation] ![Step 2](./assets/step_02_2025-01-15_14-30-45.png) --- ## Step 3 **Action:** [TODO: Select user and click 'Reset password'] ![Step 3](./assets/step_03_2025-01-15_14-31-10.png)

PSA Integration

Halo

Paste text content, then drag images individually into the rich text editor. Or convert to HTML first.

ConnectWise

Same workflow - Markdown export compatible with most documentation systems.
