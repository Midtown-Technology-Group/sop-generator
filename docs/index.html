SOP Generator

Local browser workflow capture for Halo KB drafts

Record human-led browser workflows with an unpacked extension, keep raw capture machine-local, draft Halo KB-ready HTML, and publish reviewed content through Bifrost.

View on GitHub Get Started

How It Works

1

Run Companion

Start the Python companion service on localhost. It owns local session storage, drafting, HTML export, and Bifrost publish requests.

2

Load Extension

Load extension/browser as an unpacked extension in Edge or Chrome so the technician can record in their normal browser profile.

3

Capture Events

The extension captures meaningful clicks, navigation, form changes with value hints, and form submits while a session is explicitly running.

4

Draft SOP

Generate a local draft from captured browser events and the machine-local house style file.

5

Review HTML

Export Halo KB-ready HTML and manually review it before publishing. Approval is an operator step, not an enforced code gate yet.

6

Publish

Send reviewed HTML to Bifrost for Halo KB publishing. The browser extension never stores direct Halo credentials.

Installation

Python Companion

# Clone the repository git clone https://github.com/Midtown-Technology-Group/sop-generator.git cd sop-generator # Install and start the local companion python -m pip install -e .[dev] python -m sop_generator init-style python -m sop_generator serve --host 127.0.0.1 --port 8765

Windows Launcher

# Default localhost companion .\Start-SOPGenerator.ps1 # Optional host and port .\Start-SOPGenerator.ps1 -HostName 127.0.0.1 -Port 8765

Browser Setup And SOP Commands

Operator Flow
# Load extension/browser as an unpacked extension in Edge or Chrome. # Start and stop a capture session from the extension popup. # Then use the session id with the companion CLI: python -m sop_generator draft <session-id> python -m sop_generator export <session-id> # Review the draft or exported HTML before publishing. $env:BIFROST_URL = "https://bifrost.example" $env:BIFROST_TOKEN = "<token>" python -m sop_generator publish <session-id> # Or publish with flags: python -m sop_generator publish <session-id> --bifrost-url https://bifrost.example --token <token>

Local Storage And Publishing

Machine-Local Capture

Raw sessions, drafts, exports, and house-style.md stay under %LOCALAPPDATA%\MTG\SOPGenerator unless an operator exports or publishes reviewed content.

Halo KB

The companion renders reviewed drafts as Halo KB-ready HTML so the article body is shaped before publishing.

Bifrost

Bifrost owns Halo authentication, article creation, image handling, retries, and returned publish status.
